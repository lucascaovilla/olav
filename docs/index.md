# Olav

> **Scaffold production-grade .NET APIs with strict Domain-Driven Design -- from zero to architecture in one command.**

Olav is a .NET CLI tool that generates production-ready API projects following strict **Domain-Driven Design (DDD)** architecture. A single command gives you a layered DDD solution, architecture tests that fail the build on violations, Docker configuration, GitHub Actions CI/CD, and an `olav.json` contract tracking your template version.

## Why Olav?

**Sir Nils Olav III** is a king penguin at Edinburgh Zoo holding the rank of Brigadier in the Norwegian King's Guard -- not earned through battle, but by showing up, standing straight, and being found correct every single time.

**Saint Olav II** unified Norway under a single set of laws. He imposed structure not to constrain -- but because structure is what makes things last.

Olav the tool borrows from both.

## Get Started

```bash
dotnet tool install --global Olav.Cli
olav new MyApi
```
